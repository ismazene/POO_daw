# Documentacion MonnegreFC

Este archivo se ha creado para incluir documentacion del proyecto.

————————————————————————————————————————————————

## Resumen
MonnegreFC es una aplicacion en Java para gestionar integrantes del MonnegreFC.

————————————————————————————————————————————————

## Clases principales
- MonnegreFC: clase abstracta base con nombre y edad.
- Jugador: categoria, dorsal y posicion.
- Entrenador: equipo y formacion.
- Masajista: titulacion y años de experiencia.
- Acompanyante: jugador asociado y parentesco.

————————————————————————————————————————————————

## Funcionalidades
1. Alta de nuevos jugadores.
2. Modificacion de jugadores existentes.
3. Alta de acompañantes para jugadores senior.
4. Consulta de jugadores por categoria.

————————————————————————————————————————————————

## Validaciones
- Control de dorsal repetido por categoria.
- Validacion del formato de formacion del entrenador.
